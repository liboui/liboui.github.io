libayre

A library to build GUIs by parsing various description file formats.
Currently it supports ARIA XML and Qt Designer UI file formats.

Ayre uses liboui as GUI library, which is a fa√ßade API interface to various UI library implementations
(at the moment only Qt 5/6).

Example

#include <base/fs>
#include <oui/oui>
#include <ayre/ui/factory>
int main(int argc, char *argv[])
{
oui::application app(argc, argv);
oui::window window;
fs::path path = base::filesystem::abs_exe_dir() / "resources/GUI/gui.xml";
std::unique_ptr<oui::widget> gui = ayre::make_ui(path, &window);
window.set_title("My app");
window.set_size(640, 480);
oui::center(&window);
window.show();
return app.run();
}
ayre::make_ui
std::unique_ptr< oui::widget > make_ui(const fs::path &path, oui::widget *parent)
Creates the internal GUI view from a UI file description in the parent widget.
